# GemLooks

A Ruby-based sorting/searching library.

## Supported functions

### Search algorithms

- [x] Linear search
- [x] Binary search
- [ ] Depth-first search
- [ ] Breadth-first search

### Sort algorithms

- [ ] Bubble sort
- [ ] Selection sort
- [ ] Merge sort
- [ ] Quick sort
- [ ] Heap sort
- [ ] Radix sort
